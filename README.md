# Stock Risk Radar

Stock Risk Radar is a Python-based financial analysis project designed to evaluate and rank global stocks by risk level.

The project analyzes 11,000+ publicly listed companies using market data such as stock symbol, market capitalization, price, and country information. The goal is to build a simple but scalable risk scoring system that can classify equities into different risk tiers.

This project is developed as a practical finance and data analysis tool using Python, pandas, and financial market data APIs.
